The description you want the build to be set to. Allows use of environment variables.
Automatically will set the follow GIT parameters if GIT is being used:
GIT_BRANCH
The name of the branch checked out.
GIT_REVISION
The last revision checked into the current branch
GIT_REVISION_SHORT
The last revision checked into the current branch (short)
GIT_AUTHOR
The person who last checked into the current branch
